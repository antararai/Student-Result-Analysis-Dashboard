**# Student-Result-Analysis-Dashboard:**
This project is an interactive dashboard built using Python and Streamlit to analyze student academic performance.

**Features:**
a)Subject-wise performance analysis
b)Grading system (O, A+, A, B+, B, C, D, F)
c)Select individual student result
d)Topper identification
e)Pass / Fail status
f)Pie chart and bar graph visualization
g)card-style UI

**Technologies Used:**
a)Python  
b)Pandas  
c)Matplotlib  
d)Streamlit  

This project demonstrates skills in data analysis, visualization, and basic full stack dashboard development.
